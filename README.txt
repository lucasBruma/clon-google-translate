Clon de Google Translate que utiliza IA para traducciones!

Este proyecto es un clon de la famosa herramienta de traducción de Google, pero con la particularidad de que utiliza inteligencia artificial para hacer las traducciones. Para lograrlo, se utiliza la API de OpenAI.


Tecnologías utilizadas

React.js
Typescript
Node.js
Express


Cómo funciona

El front-end del clon está desarrollado en React.js y Typescript. En la página principal, el usuario puede seleccionar dos idiomas distintos (por ejemplo, español e inglés), escribir un texto en el idioma origen y obtener la traducción en el idioma destino. Al dejar de escribir, el front hace una petición a una API desarrollada en Node.js y Express, que a su vez hace una petición a la API de OpenAI. Luego, la respuesta se devuelve al front-end y se muestra la traducción en la pantalla.
